PlaySections Features

Simple but powerful tools

Import Audio Files

Import from your Music Library, or from the other applications.

Set Marks

Set marks on an waveforms to divide an audio file into multiple sections.

Set Title

Set a title for each section, section names, lyrics, chord progressions.

Play Sections

Tap and jump to sections from a section list. No more A-B loop!

Set Loop

Set a section loop or song loop. You can play a specific section over again.

More Playback Options

Playback speed change, pitch shift, vocal cancel, and 7 bands EQ support.

For Musicians, Singers, Dancers, and More

PlaySections allows you to play your audio files, with a section list for each song. Set marks to divide an audio file into multiple sections. Set a title for each section and seek to any section from the section list. Set a loop, change playback speed, pitch, and use vocal cancel effects as you want. No need to use fast backward button frequently!

Many Uses

  • Set marks for intro, verses, chorus, instrumental solo
  • Set chord progressions and lyrics for each section
  • Practice of playing musical instruments, singing, and dancing
  • Analyze music structure, chords for music dictation
  • Great for language learners, transcription from audio files of meeting/conference

Tech Specs

  • Music Library Integration
  • Waveforms scroll
  • Section title edit
  • Song, Album, Artist name edit
  • Loop playback (section, song)
  • Speed control (-50% to +200%)
  • Pitch control (-12 to +12)
  • 7 bands EQ control (-24dB to +12dB)
  • FX (Vocal Cancel)
  • Share Extension, Open-in support
  • Remote control access
  • Low latency playback (Audio Unit support)
  • Background playback
  • Supports all major audio formats (AAC, MP3, WAV, AIFF...)
Notice: DRM protected content is not supported, because Apple does not allow 3rd party apps to access the files.
